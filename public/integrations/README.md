# Integration Logos

Place the logo SVG files for each integration in this folder with the following names:

- `linkedin.svg` - LinkedIn logo
- `slack.svg` - Slack logo
- `notion.svg` - Notion logo
- `google-calendar.svg` - Google Calendar logo
- `zoom.svg` - Zoom logo
- `gmail.svg` - Gmail logo
- `greenhouse.svg` - Greenhouse logo
- `workday.svg` - Workday logo

**Recommended specifications:**
- Format: SVG (preferred) or PNG
- Size: 48x48px minimum (will be scaled automatically)
- Background: Transparent
- Style: Monochrome or brand colors (will work with dark/light mode)

If a logo file is missing, a gray placeholder box will be displayed instead.
